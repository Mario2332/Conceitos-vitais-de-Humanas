# Guia do Usuário - Conceitos Vitais de Humanas

## Informações do Site

**Propósito:** Este site oferece acesso gratuito a 26 materiais de estudo em PDF sobre conceitos vitais de Humanas para preparação de vestibulares, além de permitir que visitantes entrem no grupo de espera da Mentoria Mário Machado 2026.

**Acesso:** Público, sem necessidade de login.

---

## Powered by Manus

Este site foi desenvolvido com tecnologias modernas de ponta para garantir a melhor experiência possível. O **frontend** utiliza **React 19** com **TypeScript** para máxima performance e segurança de tipos, combinado com **Tailwind CSS 4** para um design responsivo e elegante, além de componentes **shadcn/ui** que proporcionam uma interface moderna e acessível. A **implantação** conta com infraestrutura de auto-escalonamento com CDN global, garantindo velocidade de carregamento excepcional em qualquer lugar do mundo.

---

## Usando Seu Site

O site apresenta uma estrutura clara e intuitiva dividida em três seções principais. Na seção hero do topo, você encontra o título "Conceitos vitais de Humanas" com o subtítulo explicativo sobre revisão de tópicos conteudistas. Logo abaixo, há um card destacado com o título "Garanta sua vaga na Mentoria 2026" contendo um botão azul "Entrar no grupo de espera". Ao clicar neste botão, você será redirecionado para o grupo do WhatsApp da Mentoria 2026.

A seção principal do site exibe 26 cards organizados em um grid responsivo, cada um representando um conceito vital diferente. Cada card mostra um ícone de documento, o nome do conceito e um botão "Baixar PDF". Para fazer o download de qualquer material, basta clicar no botão "Baixar PDF" do conceito desejado, e o arquivo será baixado automaticamente para seu dispositivo.

## Gerenciando Seu Site

Para visualizar o site em tempo real durante o desenvolvimento, acesse o painel "Preview" na interface de gerenciamento. Se precisar fazer alterações no título ou logo do site, vá até "Settings" → "General". Para modificar o domínio, acesse "Settings" → "Domains". O painel "Code" permite visualizar e baixar todos os arquivos do projeto. Após fazer alterações significativas, use o botão "Publish" no cabeçalho para publicar a versão mais recente do site.
